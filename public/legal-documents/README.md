# Legal Documents

This directory contains PDF files that are downloaded when users click the "Stiahnuť ako PDF" button on the legal pages.

## Required PDF Files

Place the following PDF files in this directory:

1. **cookie-policy.pdf** - For the Cookies Policy page (`/cookies`)
   - Document: ZÁSADY POUŽÍVANIA COOKIES
2. **privacy-policy.pdf** - For the Privacy Policy page (`/privacy`)
   - Document: OCHRANA OSOBNÝCH ÚDAJOV
3. **operating-regulations.pdf** - For the Operating Regulations page (`/rules`)
   - Document: PREVÁDZKOVÝ PORIADOK
4. **terms-and-conditions.pdf** - For the General Terms and Conditions page (`/terms`)
   - Document: VŠEOBECNÉ OBCHODNÉ PODMIENKY

Each PDF file is served from the `/legal-documents/` path when users click the download button.
